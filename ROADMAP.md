# Roadmap

The roadmap optimizes for trustworthy studies before breadth.

## v0.2 — public foundation

- Publish the dual Codex backend and versioned evidence contract.
- Provide `doctor`, zero-call planning, immutable studies, and publication
  checks.
- Document methodology, limitations, security, contribution, and product
  boundaries.
- Ship reproducible GitHub release artifacts.

## v0.3 — safer operations

- Generate safe Markdown, HTML, and JSON reports directly from canonical
  derived evidence.
- Add separate recovery manifests and commands.
- Add an explicitly non-scientific development profile.
- Record complete backend, retry, latency, token, and version provenance.
- Expand report inference from descriptive smoke summaries to preregistered
  analysis generated directly from canonical evidence envelopes.
- Add small committed synthetic fixtures for adapter conformance.

## v0.4 — portable agent comparison

- Add one non-Codex adapter behind a capability declaration.
- Map shared and unsupported instruction surfaces explicitly.
- Pass synthetic, isolation, operational-slice, and shadow gates.
- Publish agent-specific results before any pooled comparison.

## v1.0 — stable scientific protocol

- Freeze versioned manifest and evidence schemas.
- Automate preregistration inputs, statistical tables, and paper artifacts.
- Publish a larger preregistered multi-agent study.
- Define compatibility and deprecation guarantees.
